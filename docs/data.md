# Data products and tools

## LightGen

This package constructs observer-based lightcones from [IllustrisTNG](https://www.tng-project.org/). The code is available on [GitHub](https://github.com/SoumyaShreeram/LightGen) and is pip installable as a python package (see [here](https://pypi.org/project/light-gen/)).


### Citation

If you use this code for you work, please cite the following two related papers: 

1. Shreeram et al. 2025a: [*Astronomy & Astrophysics*, Vol 707, A287 (2026)](https://www.aanda.org/articles/aa/full_html/2026/03/aa56028-25/aa56028-25.html)  (also available or [arXiv](https://arxiv.org/abs/2409.10397)).
2. J. Carlson and M. White, ApJS 190(2010)311. (preprint available at arxiv:1003.3178).


## LC-TNGX

You can download the TNG-based gas lightcone with X-ray information from the 'Supplementary Data Catalogs' section from the official IllustrisTNG webgage [here](https://www.tng-project.org/shreeram25) (link: <https://www.tng-project.org/shreeram25>). If you need any assistance with using it, please reach out to me [here](index.md#contact).
